## Contribution

If you would like to contribute to this website, please create a pull request.
Please note that `man-epita` use [Docusaurus](https://docusaurus.io/) website generator.
Thus, you should have Node.js `>=20.0` installed to preview the website.

```bash
npm install
npm run start
```
